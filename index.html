Text Color

The color property specifies the color of text

Example:

  • body {color: red;}
  • h1 {color: #00ff00;}
  • p.ex {color: rgb(0,0,255);}
    • Syntax
    color: "color-value"

    Fonts

    Sets specific font for the text

    The font property is a shorthand property for:

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family
    • Syntax
    font-family: "font (family) name"

    Hello World

    Internal specific font {font-family:'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif}

    Defines the color of the text
    • Browser tries to read the first font-family
    • If it can't, it will choose the second one
    • if that doesn't work, then it will read the third option

    Hello World

    External specific font {font-family:'Source Code Pro', monospace

    External fonts can be used in html via by adding it to head section

    Hello World

    External specific font {font-family:'Anton', sans-serif

    External fonts can be used in html via @import url() but placed in css file at the top

    Hello World

    External specific font {font-family: "Egli font"}

    External fonts can be used in html via download as well by adding them to an assets folder in VSCode and then writing a code like so in CSS:

    @font-face { font-family: "Egli font"; src: url("./Assets/ArchitectsDaughter-Regular.ttf")

    File extensions for fonts should be .ttf or .otf

    Font-Weight

    Defines the weight of the text

    You can use numeric values. They all correspond to a particular named weight:

    If the font family doesn't provide the requested weight, it will use the closest available one.

  • 100 Thin
  • 200 Extra Light
  • 300 Light
  • 400 Normal
  • 500 Medium
  • 600 Semi Bold
  • 700 Bold
  • 800 Extra Bold
  • 900 Ultra Bold

    Font-Style

    The font-style property is mostly used to specify italic text. This property has three values:

  • normal - The text is shown normally
  • italic - The text is shown in italics
  • oblique - The text is "leaning" (oblique is very similar to italic, but less supported)

    Text-Shadow

    Syntax:
    text-shadow: h-shadow v-shadow blur-radius color|none|initial|inherit;
    h-shadow Required. The position of the horizontal shadow. Negative values are allowed v-shadow Required. The position of the vertical shadow. Negative values are allowed blur-radius Optional. The blur radius. Default value is 0 color Optional. The color of the shadow. Look at CSS Color Value s for a complete list of possible color values none Default value. No shadow initial Sets this property to its default value. Read about initial inherit Inherits this property from its parent element. Read about inherit
